PS C:\Users\LILIANA\Desktop\Masiv-main\Masiv-main\comic>

PS C:\Users\LILIANA\Desktop\Masiv-main\Masiv-main\comic> npm i

PS C:\Users\LILIANA\Desktop\Masiv-main\Masiv-main\comic> npm run serve

http://localhost:8080/
